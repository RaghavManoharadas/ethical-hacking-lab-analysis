# Ethical Hacking Lab Assessment

## Overview
Hands-on penetration testing engagement in a controlled lab.

## Key Phases
- Reconnaissance (Passive + Active)
- Scanning (Nmap, NSE)
- Vulnerability Analysis
- Exploitation (MS17-010)
- Post Exploitation (Persistence, Keylogging)
- Covering Tracks

## Tools Used
- Kali Linux
- Nmap
- Metasploit
- Wireshark

## Key Findings
- SMB vulnerability (EternalBlue)
- Weak configurations
- Lack of monitoring

## Impact
- Full system compromise possible
- CIA triad affected

## Mitigations
- Patch management
- IDS/IPS
- Network segmentation
